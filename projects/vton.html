Virtual Try-On

  • Tech Stack: Python, PyTorch, TensorFlow, OpenCV, FastAPI
  • Github URL: I cannot share the code as I do not own it. Contact heravi.ahmad65@gmail.com for further information.

Virtual Try-On is a computer vision task in which the computer takes a person and a garment image as inputs. The goal is to change the clothing of the person with the given garment. The image generation could be complicated as the person can have arbitrary pose, shape, size, and the garment could have different types (t-shirt, jacket, etc.).

To tackle this problem, we used some custom GANs, but there were some challenges. Firstly, we needed the output to be high-resolution, but normal GANs cannot generate high-resolution images, and we had to solve this issue. Secondly, we had to train with an inadequate amount of data, and we had to deal with it. Finally, we needed to speed up our training procedure as it took so long to train the models, and we tackled it with distributed training.
