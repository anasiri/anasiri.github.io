Social Cataloging Website

  • Tech Stack: PHP, HTML, CSS, JavaScript, MySQL
  • Github URL: Project Link

This was a five-person group project for the 2020 web development class. The objective was to create a website similar to Goodreads.

We were required to use MySQL as the database management system, PHP as the backend, and HTML+CSS+Javascript as the frontend. We were not permitted to use any frameworks because the project was educational. Additionally, the website had to be compatible with Sadaf, a system written in PHP, to be integrated into it if necessary.

I worked on both the frontend and backend of the site.
