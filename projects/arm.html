Robotic Arm Kinematics

  • Tech Stack: Python, ROS, C++, CMake, Gazebo, rviz
  • Github URL: Project Link

This was a project for the 2021 robotics class. The objective was to use ROS to develop forward and backward kinematics for a robotic arm. The forward and backward kinematics are detailed here.

I developed the code using ROS services, so all required is to run a launch file, and then the arm can be controlled via client code.
